# 贪吃蛇游戏执行文档

## 1. 开发环境要求

### 1.1 硬件要求

- CPU: 1GHz或更高
- 内存: 512MB或更高
- 硬盘: 100MB可用空间

### 1.2 软件要求

- 操作系统: Windows 7/8/10/11
- 开发工具: Visual Studio 2017或更高版本（推荐Visual Studio 2022）
- .NET Framework: 4.7.2或更高版本

## 2. 编译步骤

### 2.1 使用Visual Studio编译

1. **打开项目**

   - 启动Visual Studio
   - 点击"文件" > "打开" > "项目/解决方案"
   - 导航到项目目录，选择 `C++Projectsnake.slnx` 文件
   - 点击"打开"
2. **配置编译选项**

   - 在工具栏中，确保解决方案配置为"Debug"
   - 确保平台选择为"x64"
3. **编译项目**

   - 点击"生成" > "生成解决方案"
   - 等待编译完成，查看输出窗口确认编译成功
4. **查找可执行文件**

   - 编译成功后，可执行文件将生成在 `x64/Debug` 目录下
   - 文件名为 `C++Projectsnake.exe`

### 2.2 命令行编译（可选）

如果您希望使用命令行编译，可以按照以下步骤：

1. **打开开发者命令提示符**

   - 点击"开始"菜单
   - 搜索并打开"Developer Command Prompt for Visual Studio"
2. **导航到项目目录**

   ```cmd
   cd d:\others\VisualStudio\C++Project\C++Projectsnake\C++Projectsnake\C++Projectsnake
   ```
3. **编译项目**

   ```cmd
   cl /EHsc C++Projectsnake.cpp /link /out:C++Projectsnake.exe user32.lib gdi32.lib
   ```

## 3. 运行游戏

### 3.1 直接运行

1. **找到可执行文件**

   - 导航到 `x64/Debug` 目录
   - 找到 `C++Projectsnake.exe` 文件
2. **启动游戏**

   - 双击 `C++Projectsnake.exe` 文件
   - 或在命令提示符中输入：
     ```cmd
     .\C++Projectsnake.exe
     ```

### 3.2 从Visual Studio运行

1. 在Visual Studio中，确保项目已打开
2. 点击"调试" > "开始执行(不调试)" 或按 `Ctrl+F5`
3. 游戏将直接启动

## 4. 游戏操作说明

### 4.1 开始界面

- **点击开始按钮**：使用鼠标点击"开始游戏"按钮
- **按空格键**：直接开始游戏
- **查看关卡说明**：界面底部显示各关卡的特点和通关条件

### 4.2 游戏界面

- **方向键↑↓←→**：控制蛇的移动方向
- **空格键**：暂停/继续游戏
- **R键**：重新开始游戏
- **ESC键**：返回主菜单

### 4.3 游戏结束界面

- **查看结果**：界面中央显示游戏结束或通关信息
- **查看得分**：显示最终得分和到达关卡
- **重新开始**：按R键重新开始游戏

## 5. 游戏规则详解

### 5.1 基本规则

- 蛇不能撞到墙壁或自身
- 每吃到一个食物，蛇的长度增加1，得分增加
- 得分根据当前关卡难度加成（关卡越高，每个食物的得分越高）
- 完成当前关卡的苹果数量要求后，自动进入下一关卡

### 5.2 关卡规则

#### 第1关

- 苹果位置固定，不会移动
- 蛇的移动速度较慢
- 需要吃到5个苹果才能进入第2关

#### 第2关

- 苹果位置固定，不会移动
- 蛇的移动速度加快
- 需要吃到7个苹果才能进入第3关

#### 第3关

- 苹果会缓慢随机移动
- 蛇的移动速度适中
- 需要吃到10个苹果才能完成游戏

## 6. 故障排除

### 6.1 编译错误

**问题**：编译时出现错误
**解决方法**：

- 确保已安装Visual Studio 2017或更高版本
- 确保项目配置正确（Debug模式，x64平台）
- 检查代码是否有语法错误
- 重新生成解决方案

### 6.2 运行错误

**问题**：双击可执行文件后没有反应
**解决方法**：

- 确保操作系统是Windows 7或更高版本
- 检查是否有足够的内存和磁盘空间
- 尝试以管理员身份运行

**问题**：游戏界面显示异常
**解决方法**：

- 确保显示器分辨率至少为1024x768
- 检查显卡驱动是否更新

### 6.3 操作问题

**问题**：方向键控制不灵敏
**解决方法**：

- 确保没有其他程序占用键盘输入
- 检查键盘是否正常工作

**问题**：游戏速度过快/过慢
**解决方法**：

- 游戏速度根据关卡自动调整，无法手动更改
